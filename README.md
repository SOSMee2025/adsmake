# ADSmake.ai 🚀

ADSmake.ai es una plataforma impulsada por IA (Google Gemini) diseñada para transformar fotos de productos simples en creativos publicitarios de alta conversión en segundos.

## ✨ Características principaless

- **Transformación con IA**: Sube una foto de tu producto y genera anuncios de nivel profesional instantáneamente.
- **Optimización de ROAS**: Creativos diseñados basados en datos de alto rendimiento para Meta Ads y TikTok.
- **Formatos Flexibles**: Soporte para formatos 1:1 (cuadrado), 4:5 (feed) y 9:16 (Stories).
- **Generación de Video**: Crea micro-videos cinemáticos a partir de imágenes estáticas.
- **Identidad de Marca**: Mantén la coherencia de tu producto con integración de logos y fotos de soporte.

## 🛠️ Tecnologías

- **Frontend**: React 19 + TypeScript
- **Styling**: Vanilla CSS (Efectos de Glassmorphism y Animaciones Premium)
- **IA**: Google Gemini 3.1 Flash / Lite
- **Build Tool**: Vite 8

## 🚀 Inicio Rápido

1. Clona el repositorio.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Configura tu API Key de Gemini en un archivo `.env.local`:
   ```bash
   VITE_GEMINI_API_KEY=tu_api_key_aqui
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## 📦 Producción

Para generar el bundle de producción:
```bash
npm run build
```
